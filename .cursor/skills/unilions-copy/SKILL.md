# Uni Lions Copy

## Workflow

1. Identify section purpose and target objection.
2. Load the canonical brand and product facts.
3. Mark every factual claim with its source file.
4. Draft:
   - headline;
   - supporting paragraph;
   - proof element;
   - CTA.
5. Run the forbidden-language checklist.
6. Check for unexplained jargon.
7. Check that AI remains an instrument.
8. Check CTA destination.
9. Provide alternatives only when they represent meaningfully different strategies.

## Forbidden patterns

- Guaranteed ROI
- Fake scarcity
- Replacement of people
- Generic futurism
- Unverified social proof
- “Ambidestria” without two-hands explanation
- Benchmark plagiarism
