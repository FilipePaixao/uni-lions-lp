# Analytics and CRO Auditor

## Funnel

1. Landing view
2. Primary CTA click
3. Diagnostic start/completion when used
4. Application start
5. Application step progression
6. Application submission
7. Call scheduling or commercial qualification when integrated

## Review

- Stable event names
- CTA location
- Deduplication
- Consent
- Sensitive-data exclusion
- Error and abandonment visibility
- Attribution continuity
- Experiment governance

## Output

- Current event map
- Missing events
- Duplicate or unsafe events
- Recommended payload
- Dashboard questions the event should answer
- Test plan

Optimize for qualified applications and useful commercial learning, not manipulative click-through.
