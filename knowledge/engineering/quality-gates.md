# Quality Gates

## Gate 1 — Fatos

- [ ] Nome da marca e produto corretos.
- [ ] Datas confirmadas.
- [ ] Carga confirmada.
- [ ] Modalidade confirmada.
- [ ] Preço e parcelamento confirmados.
- [ ] Rotas corretas.
- [ ] Bios aprovadas.
- [ ] Depoimentos autorizados.
- [ ] Estatísticas com fonte.
- [ ] LGPD e termos revisados.

## Gate 2 — Marca

- [ ] Ambidestria explicada pelas duas mãos.
- [ ] IA como instrumento.
- [ ] Sem hype.
- [ ] Sem escassez inventada.
- [ ] Sem promessa financeira absoluta.
- [ ] Visual premium e contido.
- [ ] Imagens reais prioritárias.

## Gate 3 — Engenharia

- [ ] Typecheck.
- [ ] Lint.
- [ ] Testes.
- [ ] Build.
- [ ] Sem erro no console.
- [ ] Sem request crítico quebrado.
- [ ] Sem segredo exposto.
- [ ] Sem dependência desnecessária.

## Gate 4 — Navegador

- [ ] 360 × 800.
- [ ] 390 × 844.
- [ ] 768 × 1024.
- [ ] 1024 × 768.
- [ ] 1440 × 900.
- [ ] 1920 × 1080.
- [ ] Sem overflow.
- [ ] CTA e anchors funcionam.
- [ ] FAQ funciona.
- [ ] Rotas preservam atribuição quando necessário.

## Gate 5 — Acessibilidade

- [ ] Um H1.
- [ ] Hierarquia de headings.
- [ ] Landmarks.
- [ ] Teclado.
- [ ] Foco visível.
- [ ] Alt.
- [ ] Contraste.
- [ ] Reflow em 200%.
- [ ] Reduced motion.
- [ ] Touch targets.
- [ ] Form labels e errors.

## Gate 6 — Performance

Metas de experiência:

- LCP ≤ 2,5 s no percentil recomendado de campo quando houver medição.
- INP ≤ 200 ms.
- CLS ≤ 0,1.

Metas de laboratório são sinal, não garantia de campo.

- [ ] Hero otimizado.
- [ ] Dimensões reservadas.
- [ ] Fonts controladas.
- [ ] Scripts terceiros revisados.
- [ ] Imagens responsivas.
- [ ] Sem animação pesada.

## Gate 7 — SEO

- [ ] Title.
- [ ] Description.
- [ ] Canonical.
- [ ] Open Graph.
- [ ] Social image.
- [ ] Indexabilidade.
- [ ] Links rastreáveis.
- [ ] Structured data fiel ao conteúdo.
- [ ] Favicon e identidade.

## Gate 8 — Analytics

- [ ] Eventos definidos.
- [ ] Sem duplicação.
- [ ] Sem PII.
- [ ] CTA location.
- [ ] Aplicação e diagnóstico separados.
- [ ] Consentimento respeitado.
- [ ] Teste de payload.

## Gate 9 — Personas

- [ ] Ricardo entende valor e entregáveis.
- [ ] Mariana consegue justificar internamente.
- [ ] Carlos não é excluído por jargão.
- [ ] Ana encontra profundidade.
- [ ] Eduardo encontra custo e escopo.

## Decisão

### GO

Todos os blockers resolvidos.

### GO WITH ACCEPTED RISKS

Somente riscos não bloqueadores com responsável e data.

### NO-GO

Qualquer fato, rota, formulário, privacidade, acessibilidade crítica ou conversão principal quebrada.
