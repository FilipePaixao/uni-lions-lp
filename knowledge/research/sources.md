- Documentação: `https://cursor.com/docs`
- Rules: `https://cursor.com/docs/rules`

- Plugins oficiais: `https://github.com/cursor/plugins`

Uso:

- `.mdc` para rules.
- arquivos Markdown para agents.
- `SKILL.md` para skills.
- browser para validação visual e funcional.
- Plan Mode para tarefas amplas.

A estrutura de projeto deste kit usa `.cursor/rules`, `.cursor/agents`, `.cursor/skills` e `.cursor/commands`. Caso a versão instalada do Cursor adote descoberta diferente, validar na documentação atual e ajustar paths sem alterar o conteúdo canônico.
